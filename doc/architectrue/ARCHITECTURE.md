# 音频播放器 - 架构设计文档

>前情提要： A是开发前端的， B是开发后端的， C是进行桥接架构的
---

## 1. 概述

### 1.1 项目简介
一个基于 Qt 的跨平台音频播放器，支持 MP3、WAV、FLAC 格式。

### 1.2 技术栈
| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | QML / Qt Quick | 5.12+ / 6.0+ |
| 后端 | C++ | C++11 |
| 构建 | CMake | 3.16+ |
| 音视频 | Qt Multimedia | 5.12+ / 6.0+ |

### 1.3 设计目标
- [ ] 跨平台（Linux/Windows/macOS）
- [ ] 支持常见音频格式

### 2.2 模块划分

| 模块 | 位置 | 职责 | 负责人 |
|------|------|------|--------|
| QML 界面 | `qml/` | 用户交互、界面显示 | B |
| PlayerController | `src/` | 桥接前后端、数据转换 | A + C |
| Qt Multimedia | 系统库 | 实际播放控制 | A |
| 资源文件 | `resources/` | 图标、字体等 | B |  

这个文件的架构是基于这个dev分支的，所以不要直接将东西提交到dev分支，  
请先到对应的featrue分支上进行开发  
然后最后将会由C来进行整合代码
